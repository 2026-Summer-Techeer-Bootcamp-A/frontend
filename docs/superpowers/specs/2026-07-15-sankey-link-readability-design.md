# Sankey 연결선 가독성 개선 설계

## 목표

개념과 기술을 잇는 Sankey 연결선을 현재보다 조금 더 진하고 두껍게 보여, 밝은 배경에서도 흐름을 쉽게 추적할 수 있게 한다.

## 변경 범위

- 개념별 연결선 색의 알파 값을 `0.4`에서 `0.52`로 높인다.
- 연결선 색을 활용한 `1px` 수준의 은은한 외곽 강조를 추가해 체감 두께를 높인다.
- fallback 연결선도 같은 수준의 명암과 외곽 강조를 적용한다.
- 채용공고 공동출현 `value`, 노드 위치, 노드 크기, Treemap 옵션은 변경하지 않는다.

## 구현 원칙

연결선의 실제 `value`를 확대하거나 최소값으로 보정하지 않는다. 따라서 링크 간 상대 비율과 개념 노드 크기는 기존 데이터 결과를 그대로 유지한다. 스타일 값은 Sankey 옵션 생성 코드의 상수로 분리해 테스트에서 확인할 수 있게 한다.

## 검증

- Sankey 연결선 스타일 상수가 승인 값과 일치하는지 단위 테스트한다.
- 기존 개념 선별, 배치, Treemap 관련 테스트를 모두 실행한다.
- 프로덕션 빌드로 ECharts 옵션 및 타입 오류가 없는지 확인한다.
