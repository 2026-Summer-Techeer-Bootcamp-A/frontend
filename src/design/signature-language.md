# 시그니처 융합 디자인 언어 (정본)

우리 블루 토큰 + 모던 대시보드 레퍼런스(logistics/ai-crm/finance/investment) + 실데이터 dataviz.
**금지:** 에디토리얼/세리프/미니멀-austere, 무지개 색, 균등 막대, 플랫 원형 파이, 글래스모피즘, 네온 글로우 남발.

## 시그니처 무브
라이트 슬레이트-블루 셸 안에 **"반전 다크 히어로 패널"** 하나. 그 안에서 **내 기술이 우리 블루로 발광**한다.
(logistics의 라이트/다크 반전 + investment의 다크 절제 + 우리 토큰)

- 잉크 `#1c1d21` · 뮤트 `#7c7f88` · 흐린 라벨 `#a2a6b0`

- **프라이머리(우리 블루):** `#2f61b8` · hover `#28539c` · 틴트 `#eef3fb` · on-dark 밝게 `#5a86cf`
- **데이터 5색(카테고리, 쿨 4 + 웜 1):** 블루 `#2f61b8` · 틸 `#2a9db0` · 바이올렛 `#6b5bd6` · 앰버 `#d9932f` · 로즈 `#c8506a`

## 카드 / 패널
- 라운드: 카드 16px · 대형 히어로 20px · 내부 타일/칩 12px · 알약 999
- 라이트 카드: `#fff` + `1px #e2e5ec` + 소프트 섀도 `0 2px 8px rgba(20,30,60,.06), 0 1px 2px rgba(20,30,60,.05)`, hover lift
- 다크 히어로: `#12141a` + `1px #262a34`, **섀도 없음(보더가 일함)**
- 화면당 틴트 카드는 1개까지. 나머지는 흰색. 여백 20~24px, 중밀도.

## 차트 규칙
- **라인:** 2.5~3px, 라운드 캡, 쿨 색, 프라이머리 시리즈 `#2f61b8`. area = 색@18%→투명 세로 그라디언트. 다크선 = 흰/`#5a86cf` + 그레이 그라디언트.
- **막대:** 라운드 탑(6~8) 또는 캡슐. **고스트(아웃라인) 막대 + 하이라이트 1개(그라디언트 채움) + 값 pill + 가는 가이드선.** 균등 막대 금지.
- **도넛:** 두꺼운 갭 아크(14~18px) 라운드 캡, 중앙 KPI 볼드. 단일점수는 반원 게이지 + 직접 범례(라벨+%).
- **그리드:** 점선/헤어라인 `#eef1f6`(라이트)·`#22262f`(다크), 가로만. 축 라벨 12px `#a2a6b0`. 포커스점에 점선 세로 마커.
- **직접 라벨 > 범례.** 델타는 항상 캐럿 + 녹/적. 큰 숫자는 **소수점 회색 처리**, 숫자 컬럼 우정렬 tabular-nums.

## 모션 (기존 tokens.ts 준수)
- 진입 `decelerate` ≤300ms, tap = scale 0.97 + opacity, 500ms 하드캡. 장식용 바운스/회전 금지.

## 안티-AI 규칙 (강제)
1. 균등 막대·플랫 파이 금지 → 고스트+하이라이트, 갭 도넛.
2. 무지개 금지 → 카테고리 5색, 쿨 주도 + 웜 1, 카드당 강조 1회.
3. 실데이터 텍스처 필수 → 로고 타일·상태 칩·캐럿 델타·tabular-nums·소수점 회색.
4. 다이렉트 라벨 + **단일 포커스 상태**(모든 점 툴팁 금지). 점선 가이드 + 값 pill로 포커스.
5. 보더 XOR 섀도(둘 다 무겁게 금지). 라이트=소프트섀도+헤어라인, 다크=1px 보더·무섀도.

## 시그니처 화면
- **① 시장 좌표맵(/signal):** 다크 히어로 위 기술 지형. x=수요(share) × y=성장(delta, `techYearly`). 대부분 스킬=흐린 점, 내 기술=블루 발광+링, 최고기회 1개=콜아웃 pill. 좌상단 거대 커버리지%(소수 회색+녹캐럿+캡슐 게이지). 블루 단일색 밀도 히트맵 언더레이.
- **② 기술 네트워크(/signal/network):** 다크 히어로 위 force graph. 노드=수요 크기, 엣지=동반(`cooccurrence`). 내 기술=블루 발광, 나머지 채도↓. 클릭=이웃만 밝게(단일 포커스)+사이드 pill.
- 두 화면은 같은 다크 히어로·블루 워시·헤더 패턴·세그먼트+단일포커스를 공유해 "한 제품"으로 읽힌다.

## 데이터 (frontend/src/data/marketData.json)
- `skillShare.국내/국외`: `{N, asOf, items:[{tech,count,share,owned}]}` (상위 20, owned 실재)
- `techYearly`: `{years, series:[{tech, owned, shares:[4], delta}]}` — **좌표맵 축 소스**
- `cooccurrence`: `{ [hub]: {base, items:[{tech,coRate,coCount}]} }` — **네트워크 소스**
- `techs.json`: `[{tech,count}]` 220개 수요 유니버스
